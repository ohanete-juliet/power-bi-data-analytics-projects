HR Survey Responses Dashboard

Project Overview
This dashboard analyzes employee survey responses to understand workforce satisfaction, identify pain points, and support data-driven HR decision-making.

Business Objective
To help HR teams:
- Measure employee satisfaction levels
- Identify key drivers of dissatisfaction
- Improve employee engagement and retention

Tools Used
- Power BI
- Power Query
- Excel

Key Insights
- Satisfaction levels vary across departments and employee groups.
- Certain factors contribute more strongly to dissatisfaction.
- Clear patterns emerge that can guide HR improvement strategies.

Recommendations
- Address high-impact dissatisfaction factors first.
- Implement targeted engagement strategies for affected employee groups.
- Use continuous feedback to monitor improvement.
